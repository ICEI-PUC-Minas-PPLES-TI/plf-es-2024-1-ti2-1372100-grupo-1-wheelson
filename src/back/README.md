# Rotas de usuário

| Método | Rota                       | Descrição                                                          |
| ------ | -------------------------- | ------------------------------------------------------------------ |
| GET   | `/locador/{id}`                   | Recebe um locador pelo ID                                             |
| POST  | `/locador`                   | Cadastra um locador                                           |
| PUT  | `/locador/{id}`                   | Altera os dados de um locador                                           |
| DELETE  | `/locador`                   | Apaga o perfil de um Locador                                        |
